Тестовое задание - редактор маршрута на карте
=============================================

Локальная установка

```bash
git clone git@github.com:r72cccp/yandex-map-router.git
cd yandex-map-router
yarn install
bin/start_dev
```

[Деплой на Heroku](https://yandex-map-router.herokuapp.com/)

Запуск Selenium тестов

```bash
cd tests/selenium
cucumber
```
